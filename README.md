## 🟣 Featured Projects

<details open>
<summary><b>🚀 AI-Based Internship Recommendation System — PM Internship Scheme Portal</b></summary>
<br/>

Developed a lightweight, rule-based AI recommendation system for the **PM Internship Scheme portal** as part of **Smart India Hackathon (SIH)**. The engine matches candidate profiles against a curated dataset of real internships using weighted scoring across skills, sector preference, and location — returning the top 3–5 personalized matches ranked by score.

| Attribute | Detail |
|---|---|
| **Stack** | Java · Spring Boot · Weighted Scoring Engine · Session-Based Auth |
| **Scale** | 48 real internship listings across multiple sectors |
| **Performance** | Real-time scoring and ranking on candidate submission |
| **Security** | Session-based authentication for candidate data handling |
| **Impact** | Built for first-generation learners & low-digital-literacy users across rural and urban India |
| **Repository** | [github.com/Jhanani85](https://github.com/Jhanani85) |

Captures candidate inputs — education, skills, sector, and location — and returns ranked recommendations through a **mobile-responsive**, **6-language UI** (English, Hindi, Tamil, Telugu, Kannada, Malayalam), designed specifically to lower the barrier to entry for underserved applicants.

</details>
